# Modifier

Basic utility Modifiers.

```kotlin
implementation("io.daio.wild.modifier:<version>")
```

Modifiers are included with any component dependency but you can use them on your own if you wish.

| Platform   | Available |
|------------|-----------|
| CMP        | ✅         |
| Android Tv | ✅         |
